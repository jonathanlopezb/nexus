# Nexxus 2.0 | E-commerce del Futuro

Este es el repositorio central del proyecto Nexxus 2.0, una plataforma de e-commerce urbana impulsada por IA.

## Estructura del Proyecto

-   **/nexxus-frontend**: Aplicación Next.js (App Router, TypeScript, Tailwind CSS, Framer Motion).
-   **/nexxus-backend**: CMS Strapi (Gestión de productos, contenidos y lógica de negocio).
-   **prop.html**: Prototipo interactivo en alta fidelidad (Epañol).

## Roadmap

- [x] Prototipo Base Localizado (ES)
- [ ] Integración de CMS Strapi con Frontend
- [ ] Sistema de Lealtad "Urban Levels"
- [ ] Navegación Neural Asistida por IA (Sorenexus)
- [ ] Checkout sin fricción (Addi, Sistecredito)

## Despliegue

El frontend se despliega automáticamente en **Vercel** al hacer push a `main`.
El backend se despliega en **Heroku/Render** o similar.
